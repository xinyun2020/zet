# Hello

This is an example skill template showing the basic structure.

A template is a markdown file with YAML frontmatter that defines:
- `type:` — what gets generated (skill, agent, or rule)
- `description:` — one-line summary (required for skills)

Run `zet generate` to build this into a deployed skill.
Run `zet validate` to check all templates are well-formed.
